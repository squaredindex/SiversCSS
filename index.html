Sivers CSS

SiversCSS is a tiny bit of CSS with no classes that just uses semantic HTML5 elements to give a very minimalist style for simple articles/blogs.

Heavily stolen from inspired by Derek Sivers, who's been practising Brutalist Web Design since before it was a thing.

The rest of the content that follows is taken from Derek Sivers website to give an example of the styles.

Yes my site is plain on purpose. It’s my minimalist desire for only what’s needed.
